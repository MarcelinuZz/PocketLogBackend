====================================
  PocketLog Backend — Install Guide
====================================

Prerequisite:
  - Docker Desktop (sudah include Docker Compose v2)

1. Jalankan semua service sekaligus:

   docker compose up --build

   Atau dengan Docker Watch (hot-reload saat kamu edit kode):

   docker compose watch

2. Selesai! Semua service dan database MySQL akan otomatis berjalan.

Port yang dibuka:
  :3000  → API Gateway
  :3001  → Auth Service
  :3002  → User Service
  :3003  → Email Service
  :3004  → Wallet Service
  :3005  → Category Service
  :3006  → Transaction Service
  :3007  → Reminder Service
  :3306  → MySQL (bisa diakses dari host untuk debugging)

Perintah berguna:
  docker compose up --build    → Build ulang & jalankan semua
  docker compose watch         → Jalankan dengan hot-reload
  docker compose down          → Stop & hapus container
  docker compose down -v       → Stop & hapus container + data MySQL
  docker compose logs -f       → Lihat log semua service
  docker compose logs -f auth  → Lihat log service tertentu
  docker compose ps            → Cek status semua container

Catatan:
  - Database MySQL akan otomatis di-init dengan semua schema saat pertama kali `up`.
  - Upload avatar tersimpan di Docker volume `auth_public` (persist meski container restart).
  - Untuk mengubah environment variable (SMTP, Google OAuth, dll),
    edit langsung di docker-compose.yml atau buat file docker-compose.override.yml.

====================================
  SQL (untuk referensi manual)
====================================
File SQL schema ada di folder init-db/ dan juga di root proyek.
Untuk default category Transfer sudah otomatis di-insert via init-db/04_pocketlog_category.sql.
